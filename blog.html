Event Bubbling is a concept in the DOM (Document Object Model).It happens when an element receives an event.It can Transmitted or propagated to bubbles up to their parent.A bubbling event goes from the target element straight up.To stop the event bubbling,it can be achieved by the use of the event stopPropagation() method.

Callback function is a function which is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors
